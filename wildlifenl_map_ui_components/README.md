# wildlifenl_map_ui_components

**WildLifeNL Map UI Components** – gedeelde UI voor kaart/overview (kleuren, knoppen, overlays, responsive) voor Wild Rapport en andere WildLifeNL-apps.

## Gebruik

```yaml
dependencies:
  wildlifenl_map_ui_components:
    path: packages/wildlifenl_map_ui_components
```

```dart
import 'package:wildlifenl_map_ui_components/wildlifenl_map_ui_components.dart';

// WildLifeNLColors, WildLifeNLTextTheme, context.responsive, ErrorOverlay, BrownButton, etc.
```
